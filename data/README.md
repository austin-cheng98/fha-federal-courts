# `data/` — datasets shipped with the FHA federal-courts study

The shipped inputs reproduce the paper's tables and figures with no network access or secret:

- `external/housing_panel.csv` — real Census outcome panel (Black-White dissimilarity; includes an HMDA denial-rate column not used as an estimated outcome).
- `processed/paper_corpus.jsonl` — 751 canonical-NOS-443 full-text opinion clusters.
- `raw/bulk_fha_cases.jsonl` — 757 canonical-NOS-443 opinion-cluster metadata rows.
- `validation/excluded_non_nos443.jsonl` — 180 rows from the original candidate snapshot excluded by exact NOS normalization.
- `validation/` — the 93-opinion enriched/choice-based hand-coded overlap (30 double-coded) plus the extraction-validation and robustness outputs (reproduce with `scripts/score_goldset.py`; see §4.6 of the paper).

Everything else the analysis uses (case-feature tables, FEII/estimation panels, doctrine maps, synthetic data) is regenerated by the scripts and not committed — see [Regenerated by the pipeline](#regenerated-by-the-pipeline-not-shipped). Heavy or secret inputs (the ~50 GB CourtListener bulk dump, synthetic corpora, intermediate raw JSONL, logs, API tokens) are excluded — see [Regenerating from scratch](#regenerating-from-scratch).

Paths are repo-relative. Row counts are `wc -l` (CSV counts include the header). Court opinions are U.S. government works in the public domain.

---

## `external/housing_panel.csv` — real outcome feasibility panel (37 lines, 36 circuit-year rows)

Census ACS Black-White residential dissimilarity and HMDA mortgage denial rates, aggregated to circuit-year. The input is restricted to non-overlapping ACS five-year vintages; after joining to the substantive opinion corpus, only eight cells remain, all in 2022. This is a feasibility diagnostic rather than an estimable panel regression.

| column | meaning |
|---|---|
| `circuit` | federal circuit (`1`-`11`, `DC`) |
| `year` | 2012, 2017, or 2022 (non-overlapping ACS five-year vintages) |
| `dissimilarity_index` | County-level Census ACS Black-White dissimilarity (0-1), population-weighted to circuit — the estimated outcome |
| `denial_rate` | HMDA loan-denial rate (may be blank where HMDA coverage is thin); effectively single-year, not used as a within-unit outcome |

**Provenance:** harvested by `src/fha/housing.py` from the Census ACS API and HMDA. ACS variables are B03002_001E (total), B03002_003E (non-Hispanic White alone), and B03002_004E (non-Hispanic Black alone). Regenerate with `scripts/run_pipeline.py --source live --real-housing` (needs a free Census API key); the loader defaults to 2012, 2017, and 2022. Counties with zero or missing Black/White denominators have an explicit missingness audit and are not assigned a dissimilarity value.

## `processed/paper_corpus.jsonl` — canonical full-text district-court opinion corpus (751 lines)

The 751 of 757 canonical NOS-443 district-court clusters that carry recoverable full text. One JSON object per line.

Key fields: `cluster_id`, `case_name`, `court_id`, `circuit`, `court_level`, `date_filed`, `year`, `docket_id`, `nature_of_suit`, `judges`, `precedential_status`, `citations`, `opinion_ids`, `text`, `text_len`, `text_source`, `source`. Circuit is a geographic aggregation key, not an indication that the opinion was issued by a circuit court.

**Provenance:** CourtListener opinion clusters filtered to exact/description-equivalent NOS-443, text backfilled from the CourtListener bulk dump. The original candidate snapshot contained 937 rows; `scripts/normalize_frozen_corpus.py` removes malformed/non-443 values and writes the audit file. All 757 retained clusters are district-court observations in the frozen snapshot. This is the single file `scripts/make_paper_assets.py` reads to regenerate the paper assets.

---

## `raw/bulk_fha_cases.jsonl` — canonical population metadata (757 lines)

One JSON object per canonical NOS-443 opinion cluster, same schema as `paper_corpus.jsonl` (`text` is not required in this metadata file). The original 937-row candidate snapshot is auditable in `validation/excluded_non_nos443.jsonl`; the canonical 757-row file is the retrieval denominator used by the paper.

**Provenance:** CourtListener cluster metadata filtered to NOS-443 by `src/fha/classify.py` / `scripts/bulk_ingest.py`.

## Regenerating from scratch

Excluded from the repo (large, regenerable, or secret): the ~50 GB CourtListener bulk dump, synthetic corpora, intermediate raw JSONL, logs, API tokens, and the manuscript files. All commands run from the repo root.

### 0. Environment
```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .
```
Python 3.11+ (tested through 3.14). No GPU required.

### 1. Normalize the frozen candidate snapshot
```bash
python scripts/normalize_frozen_corpus.py
```
This keeps only exact/description-equivalent NOS-443 values, rewrites the two released corpus files, and records exclusions in `data/validation/excluded_non_nos443.jsonl`.

### 2. Regenerate all synthetic data (no secrets, no network)
```bash
python scripts/run_pipeline.py --source synthetic
```
Re-seeds `src/fha/synth.py` and rewrites, identically every time, the synthetic ground truth, corpus, housing panel, and every synthetic feature/panel/doctrine table. Re-recovers `BETA_TRUE` as the validation check (mirrored by `tests/test_pipeline.py`).

### 3. Re-harvest the real opinion corpus from CourtListener
Get a free token (<https://www.courtlistener.com/help/api/rest/>) and supply it via `export COURTLISTENER_TOKEN=...` (or a `config/.cl_token` file you create). The token raises rate limits and unlocks full opinion text (anonymous access returns snippets only).
```bash
python scripts/collect_live.py --max-pages 60 --out data/raw/live_corpus.jsonl   # resumable
python scripts/run_pipeline.py --source existing
```
To rebuild the population denominator and backfill text from a local bulk dump, use `scripts/bulk_ingest.py`, `scripts/fetch_bulk_text.py`, and `scripts/fetch_bulk_text_from_dump.py`.

### 4. Regenerate the real housing outcome feasibility panel (Census + HMDA)
Get a free Census API key (<https://api.census.gov/data/key_signup.html>), supply it via `export CENSUS_API_KEY=...`, then:
```bash
python scripts/run_pipeline.py --source live --max-pages 60 --real-housing
```
Pulls ACS dissimilarity + HMDA denial rates live and rewrites `external/housing_panel.csv`. The default loader keeps only non-overlapping vintages 2012, 2017, and 2022; it records zero-denominator exclusions instead of silently treating them as zero segregation.

### 5. Regenerate the paper's figures and tables
```bash
python scripts/make_paper_assets.py   # all paper figures/tables from paper_corpus.jsonl
python scripts/analyze_real.py        # real case-level + circuit-year analysis
python scripts/district_demo.py       # shows the district-by-year design is underpowered
```

### Secret resolution order
For both the CourtListener token and the Census key: explicit argument → environment variable → on-disk key file (`config/.cl_token` / `config/.census_key`). Nothing in the pipeline prints a secret.
